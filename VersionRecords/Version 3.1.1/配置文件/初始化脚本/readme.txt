ElasticQuartz.sh脚本用于替换之前的helpContentQuartz.sh，keywordQuartz.sh，roomPriceQuartz.sh，roomScoreQuartz.sh，roomSearchQuartz.sh等5个脚本

调用ElasticQuartz.sh时需要传递参数，如： ElasticQuartz.sh "keyword"
参数共有6种可选值：

help                 同步更新帮助文档
keyword              同步更新单词联想
roomPrice            同步更新房源显示价格
roomScore            同步房源的评分值
roomSearch           同步房源信息至搜索引擎
all                  同步以上全部数据
